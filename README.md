# Unit Deliverable 2 - Final Project GUI
Welcome to <strong>Fleet Commander!</strong> A prototype game where you manage a fleet of ships that explore, mine, salvage, and fight when the time comes!

So far the backend that is hooked up is tested with button in the main display where you can add and subtract resources from the resource bar!
Adding ships, removing ships, having time start, and pause are what I want to add next.
With goals to get an inventory, equiping gear, crew, selling, tooltips are my goals I will strive to add but may need to cut.

For this project we will primarily focus on getting the main gameplay systems working, so the focus on the combat portion, and node exploration will not be implemented.
